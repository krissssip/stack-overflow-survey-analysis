# Stack Overflow Survey Analysis

## 📊 Overview
This project performs exploratory data analysis (EDA) on the Stack Overflow Developer Survey dataset to uncover trends in technology usage, salaries, and remote work patterns.

## 🎯 Objective
The goal of this analysis is to understand key factors influencing developer salaries, identify popular technologies, and explore remote work trends.

## 🛠 Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📂 Project Structure
- notebooks/stack_overflow_survey_analysis.ipynb – main analysis notebook

## 🔍 Analysis Steps
- Data loading and initial exploration
- Data cleaning and handling missing values
- Exploratory data analysis (EDA)
- Technology usage analysis
- Remote work analysis
- Salary analysis by country and experience
- Education vs salary comparison

## 📈 Key Insights
- Python is used by ~37% of respondents
- Over 10,000 developers work fully remotely
- Salary varies significantly depending on country
- Experience has a stronger impact on salary than education
- Remote work is a common trend among developers

## 💡 Business Value
- Helps understand current trends in the tech industry
- Identifies in-demand technologies for developers
- Provides insights into salary drivers for better career decisions
